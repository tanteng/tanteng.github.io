php合并数组的两种方法：+和array_merge()的区别

Oct 11, 2012 in DEVELOP • PHP
2 min read

PHP中两个数组合并可以使用+或者array_merge，但之间还是有区别的，而且这些区别如果了解不清楚项目中会要命的！

主要区别是两个或者多个数组中如果出现相同键名，键名分为字符串或者数字，需要注意

1）键名为数字时，array_merge()不会覆盖掉原来的值，但＋合并数组则会把最先出现的值作为最终结果返回，而把后面的数组拥有相同键名的那些值“抛弃”掉（不是覆盖）

2）键名为字符时，＋仍然把最先出现的值作为最终结果返回，而把后面的数组拥有相同键名的那些值“抛弃”掉，但array_merge()此时会覆盖掉前面相同键名的值

需要注意的是数组键形式 ‘数字’ 等价于数字

<?php $a = array(‘a’,‘b’); $b = array(‘c’, ’d’); $c = $a + $b; var_dump($a);echo ‘


’; var_dump(array_merge($a, $b)); echo ‘

’;

$a = array(0 => ‘a’, 1 => ‘b’); $b = array(0 => ‘c’, 1 => ‘b’); $c = $a + $b; var_dump($c);echo ‘

’; var_dump(array_merge($a, $b)); echo ‘

’;

$a = array(‘a’, ‘b’); $b = array(‘0’ => ‘c’, 1 => ‘b’); $c = $a + $b; var_dump($c);echo ‘

’; var_dump(array_merge($a, $b)); echo ‘

’;

$a = array(0 => ‘a’, 1 => ‘b’); $b = array(‘0’ => ‘c’, ‘1’ => ‘b’); $c = $a + $b; var_dump($c);echo ‘

’; var_dump(array_merge($a, $b)); echo ‘

’;

$a = array(‘aa’ => ‘a’, ‘bb’ => ‘b’); $b = array(‘0’ => ‘c’, ‘bb’ => ‘dd’); $c = $a + $b; var_dump($c);echo ‘

’;//+会保留前面出现的值，抛弃重复键名的值 var_dump(array_merge($a, $b));//array_merge()会覆盖前面相同键名的值 ?>
