php 对一组单词进行所有可能情况的排列组合

Nov 4, 2012 · 1 minute read
DevelopPHP

有一个数组每个元素是长度不确定的单词：

$words = array('china','mother','father','hello','welcome');
这个数组元素个数不确定，寻求用php打印这些元素的所有排列组合算法 最好能控制组合长度，比如getCombines($words,$len=0)，这个$len就是控制组合有几个元素组成的，如果为0则打印所有组合 
$words = array('china','mother','father','hello','welcome');
$res = array();
getCombines($words, 3);
print_r($res);
function getCombines($arr, $len=0, $str="") {
    global $res;
    $arr_len = count($arr);
    if($len == 0){
        $res[] = $str;
    }else{
        for($i=0; $i<$arr_len; $i++){
            $tmp = array_shift($arr);
            if (empty($str))
            {
                getCombines($arr, $len-1, $tmp);
            }
            else 
            {
                getCombines($arr, $len-1, $str.",".$tmp);
            }
            array_push($arr, $tmp);
        }
    }
}
