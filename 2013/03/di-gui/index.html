已知f(1)=2;f(2)=3;f(n)=f(n-1)+f(n-2);用递归和非递归方法计算F($n)的值

Mar 15, 2013 in DEVELOP • PHP
1 min read

有一道题是这样的：f(1)=2;f(2)=3;f(n)=f(n-1)+f(n-2),写一个foo($num)方法。采用递归和非递归两种方式，计算出值。

1.递归方法

<?php function foo($num){ if ($num>0){ if($num==1){ return 2; } elseif ($num==2){ return 3; } else{ return foo($num-1)+foo($num-2); } } }

echo foo(5);

2.非递归方法

function foo2($num){ if($num>0){ if($num==1){ $arr[0]=2; } elseif($num==2){ $arr[1]=3; } elseif($num>2){ $result=$arr[0]+$arr[1]; for($i=2;$i<$num;$i++){ $arr[$i]=$arr[$i-1]+$arr[$i-2]; $result=$result+$arr[$i]; } return $result; } } } echo foo(5);
