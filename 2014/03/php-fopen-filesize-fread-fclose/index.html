PHP函数:fopen,filesize,fread,fclose

Sun, Mar 9, 2014

PHP操作文件常用函数：fopen打开一个文件句柄，filesize读取文件大小，fread读取文件，fclose关闭文件操作句柄。

$file = fopen(__FILE__,'r');
$file_data = fread( $file, filesize(__FILE__) );
fclose($file);
var_dump($file_data);

程序示例是读取自身，并输出全部。
