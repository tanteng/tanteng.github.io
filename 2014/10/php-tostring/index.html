PHP魔术方法：toString

Wed, Oct 29, 2014

PHP的魔术方法toString，可以直接echo这个实例输出字符串，示例如下：

class haha
{
    public function __toString()
    {
        return $this->test();
    }

    protected function test()
    {
        return 'hehe';
    }
}

$aa = new haha();
echo $aa;exit;//注意，只有echo的时候才会输出字符串，如果var_dump还是打印这个示例的结构。

打印结果：hehe

定义了一个类aa，aa实现了一个魔术方法toString，这个toString直接调用另一个方法，然后return一个字符串，而且toString魔术方法只能返回一个字符串。

注意，只有echo的时候才会输出字符串，如果var_dump还是打印这个示例的结构。

这样echo这个实例，就是输出这个toString返回的字符串。
