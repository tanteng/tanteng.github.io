PHP定义二维数组静态变量类型

Dec 1, 2014 in DEVELOP • PHP
1 min read

PHP定义静态变量类型就是在变量前加上关键字static，如static $string=”,static $array=array();但是如何给二维数组定义静态变量类型，如static $array[‘abc’]=array()这样定义行吗？

这样定义PHP二维数组静态类型是错的，比如要定义$array[‘abc’]为静态变量，只需给$array定义成静态变量，如static $array=array().

定义静态变量是针对这个变量的，一个数组就是一个变量，无论它是一位数组还是二维数组，不存在定义数组下面的元素为静态变量，只有这个变量本身是什么类型。
