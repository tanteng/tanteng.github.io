PHP xml转换json

Dec 11, 2014 in DEVELOP • PHP
1 min read

PHP xml转换json很简单，使用自带的xml类simplexml_load_file方法读取xml文件，再用json_encode函数直接把xml转换json格式。代码示例：

$x=new xml(); $x->xml2json(); class xml { function xml2json() { $xml_object=NULL; if (file_exists(‘test.xml’)) { $xml_object = simplexml_load_file(‘test.xml’); } else { exit(‘Failed to open test.xml.’); }
$xml_json=json_encode($xml_object); echo $xml_json; } }

知识点：利用xml类simplexml_load_file方法读取xml文件，json_encode函数转换xml成json格式。
