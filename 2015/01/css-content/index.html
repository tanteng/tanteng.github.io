CSS content属性

Jan 20, 2015 · 1 minute read
Develop前端

浏览网页的时候，审查元素，发现css定义了content属性，很好奇，css content属性是什么？content属性怎么用呢？比如这一段css代码：

/* Post formats */
.genericon-standard:before {        content: '\f100'; }
.genericon-aside:before {           content: '\f101'; }
.genericon-image:before {           content: '\f102'; }
.genericon-gallery:before {         content: '\f103'; }
.genericon-video:before {           content: '\f104'; }
.genericon-status:before {          content: '\f105'; }
.genericon-quote:before {           content: '\f106'; }
.genericon-link:before {            content: '\f107'; }
.genericon-chat:before {            content: '\f108'; }
.genericon-audio:before {           content: '\f109'; }

CSS content示例

a:after{
  content: " (" attr(href) ")";
}

这个css content属性的意思是：给所有链接后面加上括号，括号里面的内容是链接地址。神奇吧！

也就是说，本来是这样一个链接：CSS content属性。加上这个css的content属性后，页面上此处链接就会变成：CSS content属性（https://blog.tanteng.me/2015/01/css-content/）

现在懂了content属性了。为了更好的理解，我们可以把content理解为内容，就是文字之类的，也就是给某个元素加上一些“文字”内容，以类为单位统一添加。

CSS  before属性

h1:before {content:url(/i/w3school_logo_white.gif)}

before属性表示该元素之前进行样式操作。比如这个代码，就是在所有h1的标题之前，用content属性显示一张图片。这样可以很方便的给某个类加上图标之类的样式。
