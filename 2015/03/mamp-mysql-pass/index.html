MAMP如何修改mysql密码

Mar 14, 2015 in DEVELOP • MYSQL • 数据库
1 min read

MAMP是OS X系统下常用的PHP集成环境，如何修改MAMP登陆mysql的密码呢？

1.打开终端

按control＋空格打开Spotlight搜索terminal.app，然后点击终端的黑色图标，这样就打开了Mac下的终端命令端口。（我习惯这样打开，其他方法也可以）

然后输入如下命令：

/Applications/MAMP/Library/bin/mysqladmin -u root -p password

接着输入你的用户名Mac密码，接着输入mysql密码，并且确认密码。这样就修改OK了。

2.进入MAMP文件夹配置文件修改mysql密码

这个时候还需要打开：/Applications/MAMP/bin/phpMyAdmin-X.X.X/config.inc.php

这个文件，找到$cfg[‘Servers’][$i][‘password’]这个地方，修改你刚才进行过修改的密码。

$cfg[‘Servers’][$i][‘auth_type’] = ‘config’; // Authentication method (config, http or cookie based)? $cfg[‘Servers’][$i][‘user’] = ‘root’; // MySQL user $cfg[‘Servers’][$i][‘password’] = ‘tanteng’;

经过这两个步骤，就正确修改了Mac下PHP集成环境MAMP的mysql密码。
