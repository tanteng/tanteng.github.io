sequel导入sql文件的时候提示错误

Sat, Mar 21, 2015

sequel 导入 sql 文件的时候提示错误：An error occurred when reading the file, as it could not be read in the encoding you selected utf8.

我的sql文件导出的时候是utf8编码，因此应该是以utf8作为编码的，再sequel新建数据库也是选择utf8编码，然后点击File－import选择sql文件，设置下面的选项Format为sql格式，Encoding为utf8，这应该没错的吧，然后点击确定开始导入，就提示以上的错误了，大意是选择的编码不正确，试了很多次都是如此。

后来干脆故意把选项的Encoding设为latin编码，也就是ISO Latin 1编码，竟然成功导入了，导入后打开数据表查看，发现中文也没有乱码。不知道为什么要这样就OK了。
