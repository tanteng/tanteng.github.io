Redis如何存取PHP多维数组

Apr 21, 2015 · 1 minute read
DevelopPHP

Redis本身不支持存取PHP数组的数据结构，但是如何存取PHP的数组呢？可以把数组序列化，以字符串的形式缓存到Redis中。

其实很多框架或封装好的Redis类中，都做好了自动转化，直接存数据就可以了。如代码：

public function testSavePHPArray() {
    $array = ['array1','array2',['array3key'=>'array3value']];
    $redis = RedisHandle::getInstance();
    $redis->set('save_php_array_test',$array);
}

这样就保存了一个PHP数组，取这个数组也很简单：

$redis->get('save_php_array_test');

也会自动把序列化的数据转化回数组的形式。要强调的是这样做之所以可以是，是因为一般封装好的Redis类中自动转换了数组，而redis本身不支持PHP数组的数据结构。
