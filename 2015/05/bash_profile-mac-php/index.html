修改bash_profile文件切换Mac默认PHP版本为MAMP

Sat, May 30, 2015

Mac自带了PHP开发环境，但是很多开发者装了MAMP集成环境，打开终端输入：which php，会显示/usr/bin/php，是系统自带的PHP版本，要切换默认的PHP版本为MAMP下的PHP，需要修改系统bash_profile并执行这个shell脚本。

打开终端，输入：

sudo nano ~/.bash_profile

然后把以下代码添加到bash_profile脚本最后面：

# Setting PATH for MAMP
PATH="/Applications/MAMP/bin/php/php5.6.2/bin:$PATH"
export PATH

如图所示：

 

然后按Control+O保存，再按Control+X退出。

这个时候别忘了执行一遍这个bash_profile脚本，否则不会生效，执行bash_profile脚本的命令是source：

source .bash_profile

这个时候再输入：which php命令，是不是就切换成了MAMP目录下的PHP版本了。
