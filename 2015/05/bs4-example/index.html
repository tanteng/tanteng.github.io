bs4(beautiful soup 4)简单抓取示例

May 12, 2015 · 1 minute read
DevelopDjango

参照bs4官方中文文档http://beautifulsoup.readthedocs.org/en/latest/照着写了个简单的例子，感觉它非常强大！！

import urllib.request
from bs4 import BeautifulSoup
import sys

url = 'https://blog.tanteng.me'
html = urllib.request.urlopen(url).read()

soup = BeautifulSoup(html)
h1s = soup.find_all('h1',class_='entry-title')

'''打印所有H1标题和链接'''
for h1 in h1s:
    #print(h1)
    print(h1.get_text())
    print(h1.a.attrs['href'])
