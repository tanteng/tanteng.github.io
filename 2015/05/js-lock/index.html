js防止按钮重复提交

May 19, 2015 · 1 minute read
Develop前端

js控制按钮防止ajax重复提交数据，按钮不是button，是a标签。

if($("body").data("applycancel")=="lock"){
  return false;
}else{
    $("body").data("applycancel","lock");
}

ajax提交成功后解除锁定：

$("body").data("applycancel","unlock");

这个简单的例子有一个地方值得注意，那就是jQuery数据缓存$.data的使用，关于这个，请参见：

http://segmentfault.com/a/1190000000626031
