Python斐波拉契数列生成器

Wed, May 13, 2015

利用Python生成器定义斐波拉契数列generator：

def fib(max):
    n, a, b = 0, 0, 1
    while n < max:
        yield b
        a, b = b, a + b
        n = n + 1

for i in fib(30):
    print(i)
