Python文件名不能和已有模块名重名

Mon, May 11, 2015

今天学习Python正则表达式，用到了re模块，新建了一个“re.py”文件写几行代码，但就是通不过编译，提示错误：’module’ object has no attribute ‘compile’。但是在命令行测试是可以的，问题在于我的文件名re.py命名不规范，和Python自带的re模块重名了，这是不允许的。
