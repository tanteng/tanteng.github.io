MongoDB和MySQL对应关系

Jun 22, 2015 · 1 minute read
MongoDB数据库

MongoDB是nosql类型的数据库，MySQL是传统关系型的数据库，它们都是数据库，然而在语法、结构上都有很大差别，不过很多东西还是可以做类比，下面是MongoDB和MySQL一些对应关系。

MongoDB和MySQL术语上的对应关系：

SQL Terms/Concepts MongoDB Terms/Concepts
database database
table collection
row document or BSON document
column field
index index
table joins embedded documents and linking
primary key

Specify any unique column or column combination as primary key.

 primary key

In MongoDB, the primary key is automatically set to the _idfield.
aggregation (e.g. group by) aggregation pipeline

See the SQL to Aggregation Mapping Chart.

 
              <h3>
                MongoDB和MySQL在增删查改语句上的对应关系：
              </h3>

              <p>
                比如创建一张表，MongoDB语句如下：
              </p>

              <pre class="lang:js decode:true ">db.users.insert( {
user_id: "abc123",
age: 55,
status: "A"

} )

              <p>
                或者
              </p>

              <pre class="lang:js decode:true ">db.createCollection("users")</pre>

              <p>
                这对应的MySQL语句为：
              </p>

              <pre class="lang:mysql decode:true ">CREATE TABLE users (
id MEDIUMINT NOT NULL
    AUTO_INCREMENT,
user_id Varchar(30),
age Number,
status char(1),
PRIMARY KEY (id)

)

              <p>
                再比如MongoDB和MySQL创建索引的方式对比：
              </p>

              <p>
                MongoDB创建索引语句：
              </p>

              <pre class="lang:js decode:true ">db.users.createIndex( { user_id: 1 } )</pre>

              <p>
                对应的MySQL语句是：
              </p>

              <pre class="lang:mysql decode:true ">CREATE INDEX idx_user_id_asc

ON users(user_id)

              <p>
                再比如增加字段，MongoDB是nosql数据库，并没有MySQL固定表结构的概念，给MongoDB的集合（对应MySQL的表）添加字段，对于MongoDB来说就是加一个field。
              </p>

              <pre class="lang:js decode:true ">db.users.update(
{ },
{ $set: { join_date: new Date() } },
{ multi: true }

)

              <p>
                第一个位置为空，说明是给集合里面每个元素都加上join_date的键值，这样相当于MySQL中一张表统一加上join_date这个字段，对应的sql语句如下：
              </p>

              <pre class="lang:mysql decode:true ">ALTER TABLE users

ADD join_date DATETIME

              <p>
                再比如最常见的操作之一，增加一条数据，MongoDB的语法：
              </p>

              <pre class="lang:js decode:true ">db.users.insert(

{ user_id: “bcd001”, age: 45, status: “A” } )

              <p>
                MySQL的sql语句：
              </p>

              <pre class="lang:mysql decode:true ">INSERT INTO users(user_id,
              age,
              status)

VALUES (“bcd001”, 45, “A”)

              <p>
                具体请见：<a href="http://docs.mongodb.org/manual/reference/sql-comparison/" target="_blank" rel="nofollow">http://docs.mongodb.org/manual/reference/sql-comparison/</a>
              </p>
