ThinkPHP扩展MongoDB Model支持getGridFS存取文件

Jun 19, 2015 · 1 minute read
DevelopPHPThinkPHP

ThinkPHP自带MongoDB Model，Model继承MongoDBModel可以操作MongoDB数据库，但只能操作基本的一些MongoDB操作，无法操作getGridFS存取文件，需要扩展Driver/Db/DbMongo.class.php文件。

在Driver/Db/DbMongo.class.php文件添加方法：

/**
 * 添加getGridFS
 * tuntun @ 2015年6月19日 for 使用MongoDB的getGridFS存取文件
 */
public function getGridFS(){
    return $this->_mongo->getGridFS();
}

在MongoDBModel.class.php文件添加方法：

/**
 * 添加getGridFS
 * tuntun @ 2015年6月19日 for 使用MongoDB的getGridFS存取文件
 */
public function getGridFS(){
    return $this->db->getGridFS();
}

然后在自定义模型中这样使用，如Class FileExcelModel extends MongoModel中，这样，在ThinkPHP中使用MongoDB的文件存储getGridFS示例：

public function getGridFS(){
    return $this->db->getGridFS();
}

//测试示例，上传当前目录一个文件返回object ID值
public function demo(){
    $id = $this->getGridFS()->storeFile(dirname(__FILE__).'/GoodsModel.class.php',array("author"=>'tanteng'));
    echo $id;
}

//从MongoDB读取文件
public function read($id){
    $id = '5583b4bb988d4d6805004081';
    $file = $this->getGridFS()->findOne(array('_id'=>new MongoId($id)));
    echo $file->getBytes(); //输出数据流
}

在tp的配置文件中：

'MONGO_HOST'=>'127.0.0.1',
'MONGO_DB_NAME'=>'send_excel',//数据库名
'MONGO_USER'=>'',//数据连接登录名
'MONGO_PWD'=>'',//数据库登录密码
'MONGO_PORT'=>'27017',//数据库端口
'MONGO_DB_PREFIX'=>'dd_',//数据表前缀

注意配置文件定义了MongoDB的数据库名，及集合（表）的前缀。这样就可以通过ThinkPHP使用MongoDB的getGridFS方法存取文件了。
