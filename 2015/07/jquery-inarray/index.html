jQuery inArray不存在返回-1

Jul 6, 2015 · 1 minute read
Develop前端

判断上传的文件名扩展名是否符合要求，同时禁用提交按钮，这里用到了jQuery的inArray函数，判断是否在js数组中，不存在返回-1.

//显示上传的文件名
$('#batch_file').live('change', function() {
  file_name = $(this).val();
  $("#file_name").html(file_name);
  fileExt = file_name.substr(file_name.lastIndexOf(".")).toLowerCase();//获得文件后缀名
  if(-1 == $.inArray( fileExt, ['.xls','.xlsx'] )){
    $("#fileExtErr").show();
    $("#send_check").prop('disabled',true);
  }else{
    $("#fileExtErr").hide();
    $("#send_check").prop('disabled',false);
  }
});

当然这是在js的层面进行判断，提交后程序也要判断一次，以保证安全。
