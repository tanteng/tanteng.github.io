PHP return 的其他用法

Jul 8, 2015 · 1 minute read
DevelopPHP

PHP 的 return 的含义大家很清楚了，几乎随处可见，但是见过在函数外使用 return 的吗？如下代码：

require_once('./bb-load.php');

bb_repermalink(); // The magic happens here.

if ($self) {
    if (strpos($self, '.php') !== false) {
        require($self);
    } else {
        require(BB_PATH . 'profile-base.php');
    } 
    return;
}

这里的 return 表示执行结束。PHP 官方手册：

如果在一个函数中调用 return 语句，将立即结束此函数的执行并将它的参数作为函数的值返回。如果在全局范围中调用，则当前脚本文件中止运行。

在许多函数中可以看到直接return，后面没有值，如：

function foo(){
  $example = 3;
  return;
}

这里return就是返回一个空值，没有实际意义，表示运行到此结束而已。

PHP 的 return 可以返回任何是对象的对象，数字、字符串、数组等等。
