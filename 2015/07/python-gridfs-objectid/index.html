Python根据ObjectId下载GridFS文档

Jul 30, 2015 · 1 minute read
DevelopPython3

使用MongoDB的GridFS存取文件，如何用Python根据ObjectId来下载指定的文档呢？这里需要用到bson.objectid模块中的方法来生成MongoDB的ObjectId.

查询语句：

file = fs.find_one({'_id': ObjectId('55b98b70498e38fdda304209')})

这里ObjectId需要引入bson.objectid才能使用。

完整代码示例：

# -*- coding:utf-8 -*-

from pymongo import MongoClient
from bson.objectid import ObjectId
import gridfs

client = MongoClient('mongodb://XXXX:27017/XX_excel')
db = client.js_send_excel
fs = gridfs.GridFS(db)

file = fs.find_one({'_id': ObjectId('55b98b70498e38fdda304209')})

with open('./excel2/' + file.filename, 'wb') as f2:
    f2.write(file.read())
