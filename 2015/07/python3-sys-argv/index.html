Python3接收命令行参数sys.argv用法

Mon, Jul 13, 2015

Python3中，使用sys.argv可以接收命令行传入的参数，需要import sys模块，sys.argv第一个参数是文件名，无需定义，第二参数开始是命令行输入的参数，以空格隔开。

举例如下，运行sys_argv.py脚本附带两个参数分别是run1和run2，运行结果如下：

>>>python3 sys_argv.py run1 run2
I'm action1
I'm action2

py示例代码：

import sys

# sys.argv接收参数，第一个参数是文件名，第二个参数开始是用户输入的参数，以空格隔开

def run1():
    print('I\'m action1')


def run2():
    print('I\'m action2')


if 2 > len(sys.argv):
    print('none')
else:
    action1 = sys.argv[1]
    action2 = sys.argv[2]

    if 'run1' == action1:
        run1()
    if 'run2' == action2:
        run2()
