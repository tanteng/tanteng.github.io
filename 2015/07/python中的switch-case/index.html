Python中的switch case

Jul 9, 2015 · 1 minute read
DevelopPython3

Python本身并没有其他语言都有的switch,case这种语言结构，那么在Python中，是如何代替switch的呢？

Python使用字典的数据类型来代替switch，发挥switch的功能，代码如下：

#coding: utf-8
from __future__ import division

def jia(x,y):
    print x+y

def jian(x,y):
    print x-y

def cheng(x,y):
    print x*y

def chu(x,y):
    print x/y

operator = {'+':jia,'-':jian,'*':cheng,'/':chu}

def f(x,o,y):
    operator.get(o)(x,y)

f(3,'+',2)

这也体现了Python大道至简的风格。
