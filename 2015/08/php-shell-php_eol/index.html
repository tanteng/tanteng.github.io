PHP在shell里面换行使用PHP_EOL，shell不认识

之类的html标记。或者使用PHP_EOL (string)常量，可以取得运行系统中换行符。linux中是\n, win中是\r\n, mac中是\r.

如shell脚本：

foreach ($back_order_list as $k=>$v) {
    echo "=======type3============".PHP_EOL;
    $sorder_goods = MMulti('sorderGoods','','DB_ORDER')->where(array('sg_order_no'=>$v['bo_order_no'],'sg_id'=>$v['bo_order_goods_id']))->find();
    $sorder_info = MMulti('sorderInfo','','DB_ORDER')->where(array('soi_no'=>$v['bo_order_no']))->find();
    //完成时间，下单时间，支付时间soi_completion_time

    $sorder_info['soi_completion_time'] = strtotime($sorder_info['soi_completion_time']);
    if(!$sorder_info['soi_completion_time']){
        $sorder_info['soi_completion_time'] = '0';
    }
