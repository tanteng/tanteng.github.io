PHP约束参数类型

Thu, Oct 29, 2015

PHP5起可以约束参数类型，在Yii2框架中使用很多，可以指定一个类的方法或者函数参数必须是某个类型，类型可以是对象，数组，字符串，整形。如果指定了参数类型，那么传入的值必须是符合类型约束的，否则会报异常错误。

在Yii2中新建一个model写个方法：

namespace common\tanteng;

use yii\base\InvalidParamException;

class Tanteng
{
    public static function hello()
    {
        echo 'Hello.';
    }

    public function testarray(array $array)
    {
        var_dump($array);
    }
}

在Action中调用：

namespace frontend\controllers;
use yii;
use yii\web\Controller;
use frontend\models\TestInstance;
use common\tanteng\Tanteng;

class TestController extends Controller
{    
    public function actionTestarray()
    {
        $testarray = new Tanteng;
        $testarray->testarray([3,3]);
        exit;
    }
}

输出正常，该示例即为打印出传入的数组参数。如果故意不传入数组类型的参数，则会显示错误：

PHP Recoverable Error – yii\base\ErrorException

Argument 1 passed to common\tanteng\Tanteng::testarray() must be of the type array, integer given, called in D:\xampp\htdocs\yii2\frontend\controllers\TestController.php on line 41 and defined

关于具体的PHP类型约束，参加官方手册：http://php.net/manual/zh/language.oop5.typehinting.php
