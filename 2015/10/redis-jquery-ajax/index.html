如何防止ajax重复提交

Oct 25, 2015 in DEVELOP • PHP
1 min read

关于防止重复提交，可以参考如下链接：

  1. 使用Redis加锁防止重复提交
  2. 防止表单重复提交的几种方式
  3. jquery的data方法用法

本文讲讲如何使用redis和jquery防止ajax重复提交，如果按钮是button，点击按钮之后，js使按钮禁用，程序那边同时也加锁即可。如果按钮是a标签，则需要使用jquery的data方法在页面上加锁，具体方法见上面第3个链接。

防止ajax重复提交，PHP和js两个地方都需要做防止重复提交的逻辑。

1.PHP防止重复提交

PHP通过Redis加锁防止重复提交，代码如下：

/** * 确认收货 */ public function confirmReceive(){ //是否ajax请求 if(!IS_AJAX){ redirect(URL_MEMBER . ‘/order’);exit; } //获取提交数据 $orderNo = I(“post.orderNo”);

if(!($this-&gt;lock(10,'confirm_receive_'.$orderNo))){
    $data['code'] = 600;//重复提交
    $this-&gt;ajaxReturn($data,'JSON');
}
//more

}

2.js页面加锁防止重复提交

如果按钮不是button，点击之后无法直接禁用，可以在页面上通过jquery的data方法给页面加锁。

$(‘#confirmYes’).on(‘click’,function(){ if($(“body”).data(“confirmReceive”)==“lock”){ return false; }else{ $(“body”).data(“confirmReceive”,“lock”); } $.ajax({ type:“post”, dateType:“json”, url:URL_MEMBER+‘/order/confirmReceive’, data:({‘orderNo’:order_no,‘orderBack’:orderBack}), success:function (data) { if(data.code!=200){ //alert(“更新订单状态失败”); window.location.href=‘/order/’; }else{ window.location.href=‘/order/’; } $(“body”).data(“confirmReceive”,“unlock”); } }); });

如此一来，就可以彻底杜绝重复提交了。
