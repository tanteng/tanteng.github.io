Python装饰器示例

Wed, Dec 9, 2015

在许多地方会发现这样的写法：

@app.route('/')
def hello_world():
    return 'Hello World!'

调用函数前加一个@是什么鬼，原来它是使用Python装饰器。根据网上的教程写一个简单的Python装饰器示例：

# coding=utf-8

# 定义一个装饰器
def mydecorator(func):
    def wrapper(*args,**kw):
        print('hi,now is:')
        return func(*args,**kw)
    return wrapper

# 使用装饰器
@mydecorator
def now():
    print('2015-12-9')

now()

运行结果：

D:\learn-python>python decorator.py
hi,now is:
2015-12-9

在now函数前加@mydecorator装饰器，它自动在前面打印一行信息。
