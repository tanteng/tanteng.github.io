CentOS 7通过yum安装MySQL

Mon, Feb 8, 2016

在CentOS 7中使用yum list mysql*，没有可以安装的MySQL版本，CentOS 7中默认使用的数据库是MariaDB，关于MariaDB，它是MySQL的一个分支，MySQL被Oracle收购有闭源的风险，因此有了一个MariaDB分支，兼容MySQL，下面是CentOS 7下通过yum安装MySQL的方式。

http://sharadchhetri.com/2014/07/31/how-to-install-mysql-server-5-6-on-centos-7-rhel-7/

也可以使用MariaDB替代MySQL，请参阅：在CentOS 7中安装MariaDB数据库替代MySQL
