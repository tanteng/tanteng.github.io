阿里云CentOS安装Redis并配置生产环境自动启动

Mar 6, 2016 · 1 minute read
Redis数据库服务器

在阿里云CentOS 7系统安装Redis服务，并配置生产环境自动启动。1.安装redis 2.生成环境启动redis的配置，并介绍CentOS下update-rc.d命令不可用的解决方案。

1.安装redis

$ wget http://download.redis.io/releases/redis-3.0.7.tar.gz

$ tar xzf redis-3.0.7.tar.gz

$ cd redis-3.0.7

$ make

2.Redis生成环境自动启动

请参考官网：http://redis.io/topics/quickstart

本文只讲一个地方，官网上设置自动启动配置用到的命令：

sudo update-rc.d redis_6379 defaults

这个命令不适用于CentOS系统，CentOS系统对应的命令是chkconfig，但需要修改/etc/init.d/redis_6379文件，在头部注释中添加一行，如下第2行：

#!/bin/sh
# chkconfig: 2345 10 90  
# Simple Redis init.d script conceived to work on Linux systems
# as it does use of the /proc filesystem.

再执行命令：

chkconfig redis_6379 on

到这里就大功告成。
