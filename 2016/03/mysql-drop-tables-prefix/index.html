MySQL批量删除相同前缀的表

Mar 24, 2016 in MYSQL • 数据库
1 min read

MySQL删除表的语法是drop table xxx，但是想要删除一个数据库中相同前缀的表，该怎么写MySQL语句呢？drop table like ‘xx_%’，drop table REGEXP …，drop语句模糊匹配和正则的形式都不支持。

以下操作都在Linux下进行：

1.查出所有相同前缀的表，用CONCAT函数拼接字符串

SELECT CONCAT( ‘drop table ‘, TABLE_NAME, ‘;’ ) FROM information_schema.TABLES WHERE TABLENAME LIKE ‘wp%‘;

这样查询出结果如下：

MariaDB [blog.tanteng.me]> SELECT CONCAT(‘drop table ‘,TABLE_NAME,’;’) from information_schema.`TABLES` where TABLE_NAME LIKE ‘dy_%’;

+————————————–+

| CONCAT(‘drop table ‘,TABLE_NAME,’;’) |

+————————————–+

| drop table dy_admin;                 |

| drop table dy_admin_panel;           |

| drop table dy_admin_role;            |

| drop table dy_admin_role_priv;       |

| drop table dy_announce;              |

| drop table dy_attachment;            |

把内容粘贴到记事本，使用查找替换功能去掉“|”，再复制内容到Lunux下新建的如xx.sql文件中。

2.执行sql文件

[root@iZ94r80gdghZ tanteng]# ls

drop_tables.sql  test.sql

[root@iZ94r80gdghZ tanteng]# mysql -u root -p blog.tanteng.me < drop_tables.sql
