配置nginx 301跳转到www域

Wed, Mar 16, 2016

如一个网站www.tanteng.me，域名解析的时候配置www.tanteng.me和tanteng.me都指向一个A记录，这两个域名都可以访问，但是这样会有一些弊端，详见：为什么越来越多的网站域名不加 ‘www’ 前缀？

如何在nginx中配置，当访问tanteng.me的时候自动301跳转到www.tanteng.me呢？

在服务器/etc/nginx/conf.d目录的配置文件中，添加server块如下：

server {
    server_name tanteng.me;
    return 301 $scheme://www.$host$request_uri;
}

这样，当你访问http://tanteng.me的时候，就自动301跳转到https://www.tanteng.me了。
