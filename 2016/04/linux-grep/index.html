Linux命令：强大的grep命令

Apr 13, 2016 in 服务器
1 min read

grep是一个文本搜索工具，用于在Linux系统下查找文件，帮你在浩瀚的文件中匹配字符串找到你要找的文件位置，支持正则表达式，它非常强大。

grep -rnw ‘/path/to/somewhere/’ -e “pattern”

grep简单用法

grep -r ‘string’ ./dir/*

在dir目录下递归查找包含string的文件。

grep参数：

  • -r or -R is recursive,递归的
  • -n is line number, and 显示所在行数
  • -w stands match the whole word.
  • -l (lower-case L) can be added to just give the file name of matching files.
  • Along with these, --exclude or --include parameter could be used for efficient searching.

Something like below:

排除某个目录或文件：

grep –include=*.{c,h} -rnw ‘/path/to/somewhere/’ -e “pattern”

This will only search through the files which have .c or .h extensions. Similarly a sample use of --exclude:
