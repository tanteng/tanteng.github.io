那么这里的$_SERVER['CI_ENV']如果要定义在哪里定义呢？<br />

Nginx配置设置$_SERVER变量

打开一个nginx的网站配置：

location ~ .php$ { try_files $uri =404; root /usr/share/nginx/html/tanteng.me/public; fastcgi_pass 127.0.0.1:9000; fastcgi_index index.php; fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name; fastcgi_param MYENV ‘DEV’; include fastcgi_params; }

这里的fastcgi_param MYENV ‘DEV’，就是新加的一行语句。

在PHP中打印$_SERVER看看：
