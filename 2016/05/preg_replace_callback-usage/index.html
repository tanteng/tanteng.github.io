preg_replace_callback用法

May 28, 2016 in DEVELOP • PHP
1 min read

preg_replace_callback一般用于正则替换，三个参数，第一个参数正则表达式，第二个参数回调函数，第三个参数是要替换的内容。

看一个例子，这两个方法是在一个类里面：

//test preg_replace_callback public function preg() { $content = ‘I want {water} and {food}’; $new = preg_replace_callback(‘/{(\w+)}/‘, [$this, ‘replace’], $content); dump($new); }

//正则替换callback public function replace($matches) { $replace = [ ‘water’ => ‘travel’, ‘hehe’ => ‘coding’, ]; if (isset($replace[$matches[1]])) { return $replace[$matches[1]]; } return $matches[0]; }

结果是：

I want travel and {food}

1.第二个参数是回调函数，有多种写法，可以写成闭包，也可以指定一个方法名，在类里面也可以如示例所示。

2.回调函数接收一个参数，表示匹配的内容，如示例中的replace方法，打印这个$matches的内容：

array:2 [ 0 => “{water}” 1 => “water” ]

array:2 [ 0 => “{food}” 1 => “food” ]

$matches[0]表示匹配到的完整内容，包含正则表达式的部分，$matches[1]匹配到具体的内容，注意这里的差别。

3.回调函数返回值就是表示把匹配到的内容替换成新的内容，示例中匹配到的内容给替换了，没匹配到的还是原样。
