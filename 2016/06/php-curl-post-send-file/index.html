PHP Curl 上传文件

Thu, Jun 30, 2016

记录一个 curl 示例，代码如下：

function curl($file, $targetUrl)
{
    if (function_exists('curl_file_create')) { // php 5.5+
        $cFile = curl_file_create($file);
    } else { //
        $cFile = '@' . realpath($file);
    }
    $post = array('filename' => $cFile);
    $ch = curl_init();
    curl_setopt($ch, CURLOPT_URL, $targetUrl);
    curl_setopt($ch, CURLOPT_POST, 1);
    curl_setopt($ch, CURLOPT_POSTFIELDS, $post);
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
    $result = curl_exec($ch);
    curl_close($ch);
    return $result;
}

检测 curl_file_create 函数是为了兼容 PHP 以前版本。CURLOPT_RETURNTRANSFER 参数表示 curl_exec 返回接口原始内容。CURLOPT_POSTFIELDS 是 post 的内容。
