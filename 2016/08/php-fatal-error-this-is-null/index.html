PHP Fatal error: $this is null

Thu, Aug 18, 2016

在 PHP 中，一个类的静态方法内部调用自身类的其他方法时必须用 self::method() 的形式，即使静态方法内部方法调用其他方法也是如此，如果使用 $this->method() 的形式会报错：Fatal error: $this is null in … xx.php on line xx.

正确的一个示例：

/**
 * 更新配置后全量更新评分
 * @return bool
 */
public static function updateAllHabo()
{
    $habos = self::all();
    foreach ($habos as $habo) {
        self::updateHaboScore($habo);
    }
    return true;
}

/**
 * 更新配置后更新触达的评分
 * @param Priority $habo
 * @return bool
 */
private function updateHaboScore(Priority $habo)
{
    $habo->definition_weight_score = self::definitionWeightScore($tagUserNums = 10000);
    $habo->click_rate_weight_score = self::clickRateWeightScore($habo->click_rate);
    if ($habo->click_rate < Configure::getConfigure(1, 'click_rate_threshold')) {
        //TODO 点击率＜点击率阀值设定，自动下线
    }
    $habo->total_score = self::totalScore($habo);
    return $habo->save();
}

这其实是一个面向对象的基础问题，不过今天使用的时候竟然没有注意。

Use $this to refer to the current object. Use self to refer to the current class. In other words, use $this->member for non-static members, use self::$member for static members.

因为 $this 是调用一个对象，对象必须 new 初始化，而静态方法是常驻在内存中的，所以 $this is null，并不存在，而应该使用 self 关键字。
