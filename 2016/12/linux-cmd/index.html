Linux 中执行命令 ; 和 && 的区别

Wed, Nov 30, 2016

在 Linux 中经常使用到一个命令，如 make && make install，这里也可以使用 make ; make install，那么在 Linux 中执行命令 ; 和 && 有何区别？

&& 表示第一个命令执行成功才能执行后面的命令；

; 表示无论第一个命令执行是否成功都执行后面的命令；

还有 ||，cmd a || cmd b，表示前者执行失败了，再执行后者；
