Go 搭建 WebSocket 服务端例子

Feb 27, 2017 · 2 minute read
DevelopGo

WebSocket 是一个基于 HTML5 规范的，可以实现客户端和服务器双向通信的协议，弥补了 HTTP 协议只能客户端主动发请求，服务器响应，而服务器不能给客户端推送数据的不足。

WebSocket 握手

GET /chat HTTP/1.1
Host: server.example.com
Upgrade: websocket
Connection: Upgrade
Sec-WebSocket-Key: x3JJHMbDL1EzLkh9GBhXDw==
Sec-WebSocket-Protocol: chat, superchat
Sec-WebSocket-Version: 13
Origin: http://example.com

重点如下两个 header：

Upgrade: websocket
Connection: Upgrade

Upgrade 表示升级到 WebSocket 协议，Connection 表示这个 HTTP 请求是一次协议升级，Origin 表示发请求的来源。

WebSocket API

① 打开新的 WebSocket 连接

② 可选的回调，在连接出错时调用

③ 可选的回调，在连接终止时调用

④ 可选的回调，在 WebSocket 连接建立时调用

⑤ 客户端向服务器发送消息

⑥ 回调函数，接收服务器主动发送的消息

⑦ 根据接收到的消息，决定调用二进制还是文本处理逻辑

Go 搭建 WebSocket 服务端的例子

Go 搭建一个简单 WebSocket 服务端代码例子 test.go, 如下：

package main

import (
    "fmt"
    "log"
    "net/http"

    "golang.org/x/net/websocket"
)

func Echo(ws *websocket.Conn) {
    var err error

    for {
        var reply string

        if err = websocket.Message.Receive(ws, &reply); err != nil {
            fmt.Println("Can't receive")
            break
        }

        fmt.Println("Received back from client: " + reply)

        msg := "Received:  " + reply
        fmt.Println("Sending to client: " + msg)

        if err = websocket.Message.Send(ws, msg); err != nil {
            fmt.Println("Can't send")
            break
        }
    }
}

func main() {
    http.Handle("/", websocket.Handler(Echo))

    if err := http.ListenAndServe(":1234", nil); err != nil {
        log.Fatal("ListenAndServe:", err)
    }
}

运行服务端：

go run test.go

客户端建立 WebSocket 和发送消息

在 Firefox 或 Chrome 控制台，可以用以下代码发起客户端的 WebSocket 连接：

var wsServer = 'ws://localhost:1234';
var websocket = new WebSocket(wsServer);
websocket.onopen = function (evt) {
    console.log("Connected to WebSocket server.");
};

websocket.onclose = function (evt) {
    console.log("Disconnected");
};

websocket.onmessage = function (evt) {
    console.log('Retrieved data from server: ' + evt.data);
};

websocket.onerror = function (evt, e) {
    console.log('Error occured: ' + evt.data);
};

执行以上代码，得到输出日志：Connected to WebSocket server，表示 WebSocket 连接建立成功。
