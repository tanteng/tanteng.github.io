使用 Linux 命令分析 Nginx 日志统计 ip 访问数

Thu, Mar 16, 2017

统计 access.log 文件访问次数最多的 ip 地址，并取前 10 条记录。

Linux 命令如下：

cat access.log | awk ‘{print $1}’ | sort | uniq -c | sort -k1nr | head -10
