Golang 字节切片初始化

Sat, May 20, 2017

golang 中注意下面两种字节切片类型的初始化，一个是括号，一个是花括号，其中括号会对字符串内容进行类型转换。

func main() {
    test := []byte("abcd中文")
    fmt.Println(test)

    test2 := []byte{1, 3, 4, 56}
    fmt.Println(test2)
}

输出结果：

$ go run test.go

[97 98 99 100 228 184 173 230 150 135]

[1 3 4 56]
