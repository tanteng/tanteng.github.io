macOS 使用 brew 安装编译错误

Oct 6, 2017 in DEVELOP
1 min read

在使用 Mac 的 brew 安装一些工具的时候报错：checking whether the C compiler works… no，如图，这是怎么回事呢？？

搜索发现 Stack Overflow 上有一个同样问题的回答：

I faced this same problem after installing xcode; All I had to do was run gcc. This then asked me to accept the xcode license. After, it just worked fine.

说是本机没有接受 xcode 的协议引起的，输入 gcc 会提示 accept 这个协议，但是我输入 gcc 命令是正常的，没有任何提示。

后来意外发现，当我输入 clang 命令的时候，才有了答案说的 accept 的提示，于是按照提示进行操作。

再次运行 brew install 可以成功安装了。
