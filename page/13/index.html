$test = array(1 => 0, 99 => 0, 87 => 0, 45 => 0, 67 => 0, 11 => 1, 2 => 0);

用 asort 函数对其进行排序，asort($test) 结果如下（转换为 json 格式）：

{“67”:0,“2”:0,“1”:0,“45”:0,“87”:0,“99”:0,“11”:1}
