Laravel chunk 使用注意的问题 Aug 14, 2018

使用 Laravel 的 chunk 可以用来优化大结果集的查询，提供分块处理数据的方法，但是如下的例子就会有问题：

User::where(‘approved’, 0)->chunk(100, function ($users) { foreach ($users as $user) { $user->update([‘approved’ => 1]); } });

原因在于第一次查询：

select * from users where approved = 0 limit 100 offset 0;
