Python3单例模式示例 Nov 24, 2015

当我们理解了Python的__new__方法后，我们还可以利用它来做一些其他有趣的事情，比如实现设计模式中的单例模式(singleton) 。

因为类每一次实例化后产生的过程都是通过__new__来控制的，所以通过重载__new__方法，我们可以很简单的实现单例模式。

先看在python3中怎么实现单例模式：

class Singleton(object): def new(cls): if not hasattr(cls, ‘instance’): cls.instance = super(Singleton, cls).new(cls) return cls.instance

if name == ‘main‘: a = Singleton() b = Singleton() print(id(a)) print(id(b))
