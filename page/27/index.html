Chmod命令修改目录权限数字含义

Nov 10, 2015 · 1 minute read
Develop

常见的目录权限：

-rw——- (600) — 只有属主有读写权限。

-rw-r–r– (644) — 只有属主有读写权限；而属组用户和其他用户只有读权限。

-rwx—— (700) — 只有属主有读、写、执行权限。

-rwxr-xr-x (755) — 属主有读、写、执行权限；而属组用户和其他用户只有读、执行权限。

SourceTree提交代码合并到发布分支操作步骤

Nov 5, 2015 · 1 minute read
Develop

发布代码有一个发布分支，汇总所有要发布的分支，需要把各个分支合并到发布分支上来，每个提交的分支作为一个“点”，以下是SourceTree提交分支到发布分支的操作步骤：

1.最好以自己的分支再创建一个新的分支，防止操作过程有不可逆的失误。

2.找到第一次提交修改上次提交，右键重置到当前分支，选择软合并，这样就把自己的分支合并成一个“点”，然后提交代码。

3.到要合并的主分支，切换到“所有分支”，找到自己刚提交的点，右键“遴选”，再推送到远端。

Redis加锁防止重复操作的原理

Nov 2, 2015 · 1 minute read
DevelopPHP

使用Redis加锁的方式可以防止重复操作，包括表单提交，ajax等http请求，它的原理就是redis的incr命令给键自增，如果返回值不是1，就表示是重复请求，具体看代码：

/**
 * 锁定请求
 * @param int $expire 锁定时间
 * @param string $mark 自定义标记
 * @return bool
 */
protected function lock($expire=1,$mark=''){
    $url=strtolower(MODULE_NAME.'/'.ACTION_NAME);//操作地址
    $prefix='lock_';$sid=$_COOKIE[C("SESSION_OPTIONS.name")];
    $key=$prefix.md5($sid.$url.$mark);//令牌键值
    /**
     * @var Redis $redis
     */
    $redis = D('Redis');
    if($redis->incr($key)==1){
        return $redis->expire($key,$expire);
    }
    return false;
}

call_user_func与实例化的区别

Oct 30, 2015 · 1 minute read
DevelopPHP

使用call_user_func可以执行一个类的某个方法，而且可以给方法传参数，它跟实例化一个类再调用方法有什么区别呢？看下面的例子。

class A
{
    public function __Construct($a,$b,$c)
    {
        echo 'Construct'.$a.$b.$c;
    }

    public function test($a,$b,$c)
    {
        echo ' test'.$a.$b.$c;
    }
}

$a = new A(1,2,3);
$a->test(1,2,3);

call_user_func(['A','test'],1,2,3);
