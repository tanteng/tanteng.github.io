Python函数中的函数(闭包示例)

Jul 11, 2015 · 1 minute read
DevelopPython3

Python函数中也可以定义函数，也就是闭包。跟js中的闭包概念其实差不多，举个Python中闭包的例子。

def make_adder(addend):
    def adder(augend):
        return augend + addend
    return adder

p = make_adder(23)
q = make_adder(44)

print(p(100))
print(q(100))

运行结果是：123和144.
