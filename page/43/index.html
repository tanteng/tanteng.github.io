PHP 的“魔术常量”:__LINE__,__FILE__等 Nov 12, 2014

以下是在PHP开发中经常要用到的常量，称为PHP魔术常量：

__LINE__ 文件中的当前行号。

__FILE__ 文件的完整路径和文件名。如果用在被包含文件中，则返回被包含的文件名。自 PHP 4.0.2 起，__FILE__ 总是包含一个绝对路径（如果是符号连接，则是解析后的绝对路径），而在此之前的版本有时会包含一个相对路径。

__DIR__ 文件所在的目录。如果用在被包括文件中，则返回被包括的文件所在的目录。它等价于 dirname(__FILE__)。除非是根目录，否则目录中名不包括末尾的斜杠。（PHP 5.3.0中新增）

1，标题字数控制在60个字节内

wordpress默认上传图片（文件）路径是uploads目录的‘年/月’文件夹，我们可以通过upload_dir钩子改变默认上传路径，自定义上传路径。

该函数指定了uploads的目录为uploads的avatar文件夹：

function my_upload_dir($upload) { $upload[‘subdir’] = ‘/avatar’; $upload[‘path’] = $upload[‘basedir’] . $upload[‘subdir’]; $upload[‘url’] = $upload[‘baseurl’] . $upload[‘subdir’]; return $upload; }

wordpress隐秘常量SHORTINIT只加载必须的数据库操作文件 Jan 22, 2014

Here is start my loader file, not pretty but works for specific needs:
