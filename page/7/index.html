Golang 让 for 循环运行一段时间超时自动退出 Nov 18, 2017

如何控制 for 循环一段时间超时自动退出呢？思路很简单，就是在 for 循环中使用 select 监听 channel，代码如下：

package main

import ( “fmt” “time” )

func main() { timeout := time.After(time.Second * 10) finish := make(chan bool) count := 1 go func() { for { select { case <-timeout: fmt.Println(“timeout”) finish <- true return default: fmt.Printf(“haha %d\n”, count) count++ } time.Sleep(time.Second * 1) } }()

&lt;-finish

fmt.Println("Finish")

}

使用 Golang 发送邮件 Nov 16, 2017

使用 smtp 协议，只需要邮箱服务器主机和端口，和一个发件人帐号，可以使用 Golang 的 smtp 包发送邮件，以下是一个简单示例：

// SendToMail 用于发送邮件 func SendToMail(to, subject, body, mailType string) error { user := “tt@tanteng.me” password := 123456 host := smtp.tanteng.me:25 hp := strings.Split(host, “:“) auth := smtp.PlainAuth(“”, user, password, hp[0]) var contentType string if mailType == “html” { contentType = “Content-Type: text/” + mailType + “; charset=UTF-8” } else { contentType = “Content-Type: text/plain” + “; charset=UTF-8” }

msg := []byte("To: " + to + "\r\nFrom: xx邮箱中心&lt;" + user + "&gt;\r\nSubject: " + subject + "\r\n" + contentType + "\r\n\r\n" + body)
sendTo := strings.Split(to, ";")
err := smtp.SendMail(host, auth, user, sendTo, msg)
return err

}

重点看 msg 字段，也就是邮件标题，发送者，标题，内容的格式。

PHP intval 转换浮点数精度丢失问题 Oct 15, 2017

在 PHP 和其他一些语言都会存在这个问题，转换浮点数为整数的时候会出现精度丢失，如下：

<?php $n=“19.99”; print intval($n*100); // prints 1998

解决办法：

1.转换成字符串再 intval

print intval(strval($n*100)); // prints 1999

2.使用 round 函数替代 floatval

php -r “echo round(19.99*100);”
